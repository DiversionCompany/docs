## Create a repo using the webapp

You can start using Diversion without any installations!

* Click Login at the top of this page
* Sign up
* Click on Create New Repository
![New Repo](/images/create-repo.jpg)
* Give your new repo a name and click Create
![Repo Name](/images/repo-name.jpg)
* The tutorial takes you through installation instructions, if you want to work from your local machine. You can skip
those if you only want to use the webapp. Simply select the new repo from the dropdown menu at the top:
![Select Repo](/images/select-repo.jpg)
