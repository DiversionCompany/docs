---
title: 'Clone a repo'
description: 'Get an existing repo and work on it locally'
---

import { Image } from '/snippets/image.mdx'
import DvShellHowTo from '/snippets/dv-shell-how-to.mdx';

<Tabs>
  <Tab title="Desktop app">
    <Steps>
        <Step title="Clone a repo">
          Find the repo you want to clone, under "All Repositories".
          {/* <img src="/images/create-repo.jpg" alt="Create Repo" style={{width: '90%', borderRadius: '1.5rem', border: '.3rem solid #555', boxShadow: '0 0 1rem #888' }} /> */}
          Click on "Clone repo".
        </Step>
        <Step title="Choose a location">
          In the modal window, choose a folder where you want the repo to download to.
          {/* <img src="/images/repo-name.jpg" alt="Repo Name" style={{width: '90%', borderRadius: '1.5rem', border: '.3rem solid #555', boxShadow: '0 0 1rem #888' }} /> */}
          Click "Clone" to download the repo.
        </Step>
    </Steps>
  </Tab>
  <Tab title="CLI">
    <DvShellHowTo />
    `cd` to the directory where you want the repo to be (a new sub-directory will be created), then run:
      ```bash
      dv clone [repo_name]
      ```
      which will create a directory called `repo_name` and download the repo's contents into it.
  </Tab>
</Tabs>
